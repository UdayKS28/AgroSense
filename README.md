# AgroSense 🌱
**Farm Smarter. Harvest More. Waste Nothing.**

IoT-powered precision farming landing page — React + Tailwind CSS + GSAP.

---

## Run Locally
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```

## Deploy to Vercel
1. Push this folder to GitHub
2. Go to vercel.com → New Project → Import repo
3. Vite is auto-detected → click Deploy
4. Live URL ready in 60 seconds

---

## File Structure
- src/components/Navbar.jsx   — Sticky nav, mobile menu
- src/components/Hero.jsx     — Two-column hero + sensor cards
- src/components/Sections.jsx — All other sections
- src/data/content.js         — All data arrays (features, plans, testimonials)
